About Computer Art

Computer Art, also referred to as new media art, is a form of interdisciplinary creation that can be found everywhere in our modern day. Anything involving images, sound, animation, video, robotics, video games, etc. that were made with electronic media can be classified as computer/new media art.

Golden Spiral Animation

This was an animation I produced using python, and more specifically using pygame. In it we see an object that moves in a Golden Ratio spiral until it collides with the edge of the screen, then it changes shapes and color until it collides again. Each collision it changes to a different shape or image. The image is chosen randomly between 5 different photos.

Dice Website

This is a website I designed for my Intro to Computing Course. In this work, the website takes in an input of how many rounds you would like to play. If the dice add up to a value greater than 9, then the user wins $5. If the dice add up to a value less than 5 then the computer wins $5. Whoever has the most money after the amount of rounds input wins the game. I achieved this using JavaScript functions to track the information throughout the game.
